# Analytica - Content & SEO Dashboard

A modern marketing intelligence dashboard for tracking SEO performance, website traffic, and audience insights.

## Features
- Interactive data visualization using Chart.js.
- SEO keyword performance tracking.
- Multi-view sidebar navigation.
- Responsive professional layout.

## Technical Stack
- HTML5 / CSS3
- JavaScript
- Chart.js Library

## How to Run
Simply open `index.html` in any modern web browser.
