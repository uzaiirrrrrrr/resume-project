# CyberGuard - URL Security Scanner

A professional real-time security analysis tool that scans URLs for phishing indicators and SSL validity using JavaScript heuristics.

## Features
- SSL Certificate validation checks.
- Heuristic-based phishing detection.
- Real-time safety scoring.
- Glassmorphism UI design.

## Technical Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6)
- FontAwesome Icons

## How to Run
Simply open `index.html` in any modern web browser.
